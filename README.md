Weather Alarm
====
Gutty(Taiga Mikami) Original App　

朝起きるときに、いつものアラーム音は不快であるため、今日の「天気」と「気温」をアラーム音の代わりに音声で提供してくれるiOSアプリケーション

## Description
 早起きするのが憂鬱な朝、もしくはお昼寝の際に利用するアプリケーション
